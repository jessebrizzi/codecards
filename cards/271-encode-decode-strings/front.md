## Problem

Create functions to encode a list of strings into a single string and decode it back to the original list.

**Example 1:**
```text
Input: ["apple", "banana", "cherry"]
Output: ["apple", "banana", "cherry"]
```

**Example 2:**
```text
Input: ["hello", "world", "123", "!@#"]
Output: ["hello", "world", "123", "!@#"]
```

## Problem

Determine if a string containing `'('`, `')'`, `'{'`, `'}'`, `'['` and `']'` is properly nested and balanced.

**Example 1:**
```text
Input: "{[()()]}"
Output: true
```

**Example 2:**
```text
Input: "{[()]"
Output: false
```

**Example 3:**
```text
Input: "([)]"
Output: false
```

**Rules:**
- Each opening bracket must have a matching closing bracket
- Brackets must be properly nested
- Empty strings are considered valid
